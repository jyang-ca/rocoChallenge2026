# Data notes

- Large datasets are not tracked in git.
- HDF5 demos live on Hugging Face: `rocochallenge2025/rocochallenge2025`.
- Default paths used by tools:
  - Downloads/cache: `data/cache`
  - Exports (PNG/MP4/CSV): `data/exports`

Use `scripts/roco_hdf5_viewer.py` to pull/inspect/export without committing data.
